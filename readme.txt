Wordpress Plugin for Tamina.
This plugin lets you embed a widget for instant booking of appointments into your website.
For more information visit: https://tamina.cc/


== Description ==
 You can simply add Tamina as a widget into wordpress.
 All you need for this is a place on Google Places / Google Business.
 You can click on your admin page on Settings->Tamina to configure your widget.

Modes:

* popup
    The simple mode is **popup**. This mode only adds a social button,
    that opens a popup window when clicked.
* iframe
    The more common mode is **iframe** . Here a frame is embedded as a widget,
    showing Tamina directly on your page.
